## Changelog Overview

JetBack releases updates regularly to enhance flight monitoring, refund claiming, and user experience. Check this page for the latest version history, new features, improvements, and bug fixes. You receive notifications for major updates through the app.

<Callout kind="info">
  Enable automatic updates in your app settings to get the latest features and security improvements immediately.
</Callout>

## Recent Updates

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>
## New Features

- Added support for 15 additional airlines, including budget carriers like Spirit and Frontier
- Introduced real-time price drop alerts with estimated credit amounts before claiming

## Improvements

- Enhanced fare detection accuracy by 25% using improved airline API integrations
- Optimized battery usage during background monitoring on iOS and Android

## Bug Fixes

- Fixed occasional delays in claim submission notifications
- Resolved issue where duplicate bookings caused multiple claims
</Update>
